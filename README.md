# Frontend Mentor — Space Tourism Website (Challenge)

This repository contains a solution for the Frontend Mentor "Space Tourism Website" challenge.

## The challenge
Recreate the provided design and build a responsive multi-page site that follows the given mockups. The goal is to practice layout, responsive design, and component structure using Next.js.

## Demo
- Live demo: [Upload on Vercel](https://goota-space-tourism.vercel.app/)

## Built with
- Next.js 12
- React 18
- TypeScript
- CSS Modules
- classnames

## Project structure
- pages/ — Next.js pages
- public/ — static assets (images, icons)
- components/ — UI components
- styles/ — global and component styles

## Notes
- Follow the original Frontend Mentor designs for layout, spacing and responsive breakpoints.
- Recommended Node.js version: 16.10.0 or higher (up to 20)
- Use `pnpm` as the package manager.

## Author
Solution created for learning and portfolio purposes as part of the Frontend Mentor challenge.
